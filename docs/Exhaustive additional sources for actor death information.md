# Exhaustive additional sources for actor death information

This report identifies **150+ additional sources** beyond your existing list for tracking actor mortality data. The most valuable new sources include free APIs from **Chronicling America**, **Trove**, **Europeana**, and **Listen Notes** for programmatic access, plus high-quality manual sources like the **Television Academy In Memoriam database**, **BFI Sight & Sound obituaries**, and the **Vanderbilt TV News Archive**. International coverage gaps can be filled through region-specific databases like **AlloCiné** (France), **Douban** (China), **Soompi** (Korea), and **FilmiBeat** (India).

## Priority recommendations by use case

**Best free sources with APIs for automation:**
- Chronicling America, Trove, Europeana (historical newspapers)
- Listen Notes, Podchaser (podcast search)
- Internet Archive (books, TV news, historical)
- Wikipedia API (film accidents list)

**Best free sources for manual lookup:**
- Television Academy In Memoriam (searchable, comprehensive)
- BFI Sight & Sound Obituaries (international, detailed)
- IBDB (theater actors with death dates)
- Vanderbilt TV News Archive (62,000+ hours of coverage)

**Best paid sources for comprehensive data:**
- NTIS Death Master File ($2,500+, API available, 85M records)
- Getty/Alamy APIs (photo metadata with death events)
- American National Biography/Oxford DNB (scholarly death records)
- GenealogyBank (316M+ obituaries)

**Best regional sources by geography:**
- UK: FreeBMD (free), GRO certificates, British Newspaper Archive
- Australia: Trove (free API), Ryerson Index, State BDM registries
- France: AlloCiné, Cinémathèque Française
- Korea: Soompi, Yonhap News Agency
- Japan: Japan Times, Nippon.com
- China: Douban (has Wikidata integration), Taipei Times (less censored)
- India: FilmiBeat, Times of India Entertainment

**Sources NOT useful for death tracking** (casting platforms that remove profiles rather than track deaths): Spotlight, Actors Access, Casting Networks, Backstage.
